# Basic Network Sniffer

**Description:**
This Python script is a basic network sniffer that captures and displays network packets.

**Usage:**
1. Install Python.
2. Clone the repository.
3. Run `python sniffer.py` to start the sniffer.

**Dependencies:**
* Python
* socket library
